Если при запуске будет ошибка связанная с хромом необходимо добавить переменную среды:
Панель управления -> Система -> Дополнительные параметры -> Переменные среды 
В разделе "Системные переменные" -> Создать
В появившемся окне^
 имя переменной - "CHROME_PATH"(без кавычек)
Значение переменной - путь до chrome.exe
По стандарту: "C:\Program Files (x86)\Google\Chrome Beta\Application\chrome.exe"

Далее "ОК" и перезагружаем ПК, после чего запускаем парсер
